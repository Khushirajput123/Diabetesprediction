
# Diabetes Prediction using Machine Learning
This project builds a **Machine Learning model to predict whether a person has diabetes** based on medical features such as glucose level, BMI, age, and blood pressure. The model implemented using Python and Scikit-learn.
# Dataset
The dataset contains *768 patient records* and *9 columns*, including medical attributes and the target variable.
# Features
. Pregnancies
. Glucose
. Blood Pressure
. Skin Thickness
. Insulin
. BMI
. Diabetes Pedigree Function
. Age
# Target
 Outcome
 0 → Non-diabetic
 1 → Diabetic
# Project Workflow
1. Data Loading – Import the dataset using Pandas.
2. Data Preprocessing – Separate input features and the target variable.
3. Data Standardization – Normalize feature values so that all variables are on the same scale.
4. Train-Test Split – Split the dataset into training and testing sets.
5. Model Training – Train a classification model using Support Vector Machine (SVM).
6. Model Evaluation – Measure model performance using accuracy score.

# Tools and Libraries
* Python
* NumPy
* Pandas
* Scikit-learn

# Objective
The goal of this project is to demonstrate the **basic machine learning workflow**, including data preprocessing, feature scaling, model training, and evaluation.

* Feature scaling using Standardization
